# 💇‍♀️ Tata's Touch - Salon Management System

A full-stack (MERN) booking and management platform designed specifically for **Tata's Touch Salon**. This application streamlines the appointment process, allowing customers to book services and admins to manage operations via a secure dashboard.

## 🚀 Live Demo
[https://tatas-touch.onrender.com/]

## ✨ Key Features
* **Customer Booking:** User-friendly form to select services, dates, and times.
* **Admin Dashboard:** Secure area to view, update, and delete customer bookings.
* **Real-time Updates:** Integration with PostgreSQL (Sequelize) for persistent data management.
* **Cloudinary Integration:** (If you used it) Secure image hosting for salon gallery.
* **Responsive Design:** Optimized for both desktop and mobile salon management.

## 🛠️ Tech Stack
* **Frontend:** React.js, CSS3
* **Backend:** Node.js, Express.js
* **Database:** PostgreSQL (Sequelize ORM)
* **Hosting:** Render (Full-stack deployment)

## 📦 Installation & Setup
1. Clone the repo: `git clone https://github.com/your-username/tatas-touch.git`
2. Install Server deps: `npm install`
3. Install Client deps: `cd client && npm install`
4. Set up `.env` with your `DATABASE_URL`.
5. Run locally: `npm run dev` (from root).
